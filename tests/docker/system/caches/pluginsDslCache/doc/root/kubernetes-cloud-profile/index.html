Definition of an agent cloud profile for Kubernetes in a project.

Example. Kubernetes Cloud Profile using default service account

project {
    // Other Project settings ...
    features {
        // Other Project Features ...
        kubernetesCloudProfile {
            id = "kube-2"
            name = "Kubernetes - 2"
            apiServerURL = "https://kubernetes.internal.mycompany.com"
            authStrategy = defaultServiceAccount()
        }

            id = "kube-2"
            name = "Kubernetes - 2"
            apiServerURL = "https://kubernetes.internal.mycompany.com"
            serverURL = "https://jetbrains.com/teamcity"
